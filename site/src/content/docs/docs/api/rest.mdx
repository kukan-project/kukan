ベースパス: `/api/v1`

認証列の記号:

- **公開** — 認証不要
- **認証** — ログインまたは API トークンが必要
- **組織** — 組織の member / editor / admin のいずれか
- **sysadmin** — システム管理者のみ

## データセット (packages)

| Method | Path | 説明 | 認証 |
|--------|------|------|------|
| GET | `/packages` | データセット一覧（検索・絞り込み対応） | 公開 |
| POST | `/packages` | データセット作成 | 組織 editor |
| GET | `/packages/{nameOrId}` | データセット詳細 | 公開 |
| PUT | `/packages/{nameOrId}` | データセット更新 | 組織 editor |
| PATCH | `/packages/{nameOrId}` | データセット部分更新 | 組織 editor |
| DELETE | `/packages/{nameOrId}` | データセット論理削除 | 組織 editor |
| POST | `/packages/{nameOrId}/purge` | 論理削除済みを完全削除 | 組織 admin |
| GET | `/packages/{id}/resources` | リソース一覧 | 公開 |
| POST | `/packages/{id}/resources` | リソース追加 | 組織 editor |
| PUT | `/packages/{id}/resources/reorder` | リソース順序変更 | 組織 editor |

### 主なクエリパラメータ（GET /packages）

| パラメータ | 説明 |
|----------|------|
| `q` | キーワード検索 |
| `offset` / `limit` | ページネーション（最大 100） |
| `organization[]` | 組織で絞り込み |
| `groups[]` | カテゴリーで絞り込み |
| `tags[]` | タグで絞り込み |
| `res_format[]` | リソース形式で絞り込み（例: `CSV`） |
| `license_id[]` | ライセンスで絞り込み |
| `state` | `active` / `deleted` |
| `my_org` | 自分の所属組織に限定 |
| `private` | 非公開データを含む |
| `sort_by` | `updated` / `created` / `name` |
| `sort_order` | `asc` / `desc` |
| `include_facets` | ファセット情報を含める |

### 例: データセット検索

```bash
curl "https://your-kukan-site/api/v1/packages?q=人口&organization[]=stat-bureau&limit=10"
```

## リソース (resources)

| Method | Path | 説明 | 認証 |
|--------|------|------|------|
| GET | `/resources/count` | アクティブなリソース数 | 公開 |
| GET | `/resources/formats` | 使われているフォーマット一覧 | 公開 |
| GET | `/resources/{id}` | リソース詳細 | 公開 |
| GET | `/resources/{id}/text` | テキストコンテンツ取得（プレビューサイズ制限） | 公開 |
| GET | `/resources/{id}/download` | ファイルダウンロード（外部 URL は 302 リダイレクト） | 公開 |
| GET | `/resources/{id}/preview` | サーバー経由プレビュー（Range 対応） | 公開 |
| GET | `/resources/{id}/pipeline-status` | パイプライン処理状態 | 公開 |
| POST | `/resources/{id}/upload-url` | プリサインドアップロード URL 発行 | 組織 editor |
| POST | `/resources/{id}/upload` | サーバー経由アップロード | 組織 editor |
| POST | `/resources/{id}/upload-complete` | アップロード完了通知・パイプライン投入 | 組織 editor |
| POST | `/resources/{id}/run-pipeline` | パイプライン手動実行 | 組織 editor |
| PUT | `/resources/{id}` | リソース更新 | 組織 editor |
| DELETE | `/resources/{id}` | リソース削除 | 組織 editor |

## 組織 (organizations)

| Method | Path | 説明 | 認証 |
|--------|------|------|------|
| GET | `/organizations` | 組織一覧 | 公開 |
| POST | `/organizations` | 組織作成 | sysadmin |
| GET | `/organizations/{nameOrId}` | 組織詳細 | 公開 |
| PUT | `/organizations/{nameOrId}` | 組織更新 | 組織 admin |
| DELETE | `/organizations/{nameOrId}` | 組織論理削除 | 組織 admin |
| GET | `/organizations/{nameOrId}/members` | メンバー一覧 | 組織 member |
| POST | `/organizations/{nameOrId}/members` | メンバー追加・ロール変更 | 組織 admin |
| DELETE | `/organizations/{nameOrId}/members/{userId}` | メンバー削除 | 組織 admin |

メンバーのロールは `admin` / `editor` / `member` のいずれか。

## 管理 (admin)

すべて sysadmin 限定。

| Method | Path | 説明 |
|--------|------|------|
| GET | `/admin/search/stats` | 検索インデックス統計（件数・サイズ） |
| GET | `/admin/search/doc/{index}/{id}` | インデックスドキュメント取得（index: `packages` / `resources`） |
| GET | `/admin/search/browse/{index}` | インデックスドキュメント一覧（ページング・検索対応） |
| POST | `/admin/reindex` | 検索インデックス再構築 |
| POST | `/admin/jobs/enqueue-all` | 全リソースのパイプライン一括予約 |
| GET | `/admin/jobs/stats` | パイプラインジョブ統計 |
| GET | `/admin/jobs` | パイプラインジョブ一覧 |
| DELETE | `/admin/data` | 全データ削除（ユーザー除く） |
| GET | `/admin/health/stats` | ヘルスチェック統計 |
| GET | `/admin/health` | ヘルスチェック結果一覧 |
| GET | `/admin/users/stats` | ユーザー統計 |
| GET | `/admin/users` | ユーザー一覧 |
| POST | `/admin/users` | ユーザー作成 |
| PATCH | `/admin/users/{userId}` | ユーザー更新 |
| DELETE | `/admin/users/{userId}` | ユーザー論理削除（セッション・トークン即時失効） |
