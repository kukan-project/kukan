## データセットの作成

1. ダッシュボードにログイン
2. 「データセット作成」をクリック
3. 必要な情報を入力

### 基本情報

| 項目 | 説明 | 必須 |
|------|------|:----:|
| 名前（URL スラッグ） | 英数字・ハイフン・アンダースコア・ピリオド | ○ |
| タイトル | 表示用のタイトル | - |
| 説明 | データセットの詳細な説明 | - |
| 組織 | データを所有する組織 | ○ |
| ライセンス | データのライセンス | ○ |
| 公開 / 非公開 | 非公開にすると組織メンバーのみ閲覧可 | - |

### タグ

タグを追加してデータセットを分類できます。
既存のタグから選択するか、新しいタグを作成できます。

## リソースの管理

リソースはデータセットに紐づくファイルや URL です。

### リソースの追加

- **ファイルアップロード** — ブラウザから直接アップロード（最大 100 MB）
- **外部 URL** — 外部サイトのファイル URL を指定

### 対応フォーマット

CSV、TSV、JSON、GeoJSON、XML、PDF、Excel など多数のフォーマットに対応しています。

アップロードされたファイルは自動的にパイプライン処理され、以下が実行されます。

- テキストファイルのエンコーディング検出・変換
- CSV/TSV ファイルの Parquet 変換（テーブルプレビュー用）
- ZIP ファイルのマニフェスト生成

## 組織の管理

組織はデータセットの所有単位です。

### メンバー管理

組織の管理者は、メンバーの追加・削除・役割変更ができます。

| 役割 | データセット閲覧 | データセット編集 | メンバー管理 |
|------|:---------------:|:---------------:|:----------:|
| メンバー | ○ | - | - |
| 編集者 | ○ | ○ | - |
| 管理者 | ○ | ○ | ○ |

## カテゴリー（グループ）

カテゴリーは組織を横断してデータセットを分類する仕組みです。
データセット作成時にカテゴリーを指定できます。

### カテゴリーのメンバー管理

カテゴリーにもメンバー管理があり、組織と同様の役割構成です。

| 役割 | カテゴリー閲覧 | データセット追加 | メンバー管理 |
|------|:------------:|:-------------:|:----------:|
| メンバー | ○ | - | - |
| 編集者 | ○ | ○ | - |
| 管理者 | ○ | ○ | ○ |
