## データセットの検索

トップページの検索バーからキーワード検索ができます。

### フィルタリング

- **組織** — データを公開している組織で絞り込み
- **カテゴリー** — データの分類で絞り込み
- **タグ** — キーワードタグで絞り込み

### 並べ替え

検索結果は関連度順のほか、更新日時順や名前順で並べ替えられます。

## データセットの閲覧

データセットページでは以下の情報を確認できます。

- **メタデータ** — タイトル、説明、作成者、ライセンスなど
- **リソース一覧** — データセットに含まれるファイルや URL
- **カテゴリー・タグ** — データの分類情報

## リソースのプレビュー

リソースページでは、ファイル形式に応じたプレビューが利用できます。

| 形式 | プレビュー |
|------|-----------|
| CSV / TSV | テーブル表示 + テキスト表示 |
| JSON / XML / TXT | テキスト表示 |
| GeoJSON | 地図表示 + テキスト表示 |
| PDF | PDF ビューア |
| Excel / Word / PowerPoint | Office Online Viewer |

## データのダウンロード

リソースページの「ダウンロード」ボタンからファイルをダウンロードできます。
外部 URL のリソースも同じボタンからダウンロードできます（元の URL にリダイレクトされます）。

## API アクセス

KUKAN は REST API を提供しています。
プログラムからデータを取得する場合は、API ドキュメントを参照してください。

```bash
# データセット一覧の取得
curl https://your-kukan-site/api/v1/packages

# データセットの検索
curl https://your-kukan-site/api/v1/search?q=人口
```
